优雅地听音乐和看番

前言

好久没写博客了，水一篇也顺便记点东西。网易云音乐最近不当人，把μs的版权都搞丢了，B站番剧的和谐程度也是每况愈下，本地听音乐与看番是大势所趋了（悲）

看番

虽然PotPlayer很有名，不过公认最好的播放器（之一）是mpv，不过这玩意儿貌似最近才出图形界面，而且也挺简陋的，再加上我用mpv主要是为了大名鼎鼎的Anime4K插件，所以我选择大佬整合的mpv-lazy。

mpv-lazy在前几个版本是需要自己注册，UI也有点复古，不过最近的一次大更新变成了exe直接安装，UI也现代多了，这里简单记点东西：

  • 右键打开菜单后按住Shift可以在操作后保持菜单一直打开

  • 老版本的mpv-lazy的字幕设置被精简掉了，现在如果字幕组有配置字幕会用字幕组的格式，一般这些格式没有大问题，我也懒得去研究新版本咋调整字幕了。如果是没有定义格式的字幕，右键工具-设置目录，修改mpv.conf对应项就可以了，备份一下我的字幕设置：

  •  ########
 # 字幕 #
 ########

 sub-auto = fuzzy
 sub-file-paths = sub;subtitles
 slang = chs,sc,zh,chi,zho

 sub-font = "Microsoft YaHei"
 sub-font-size = 50
 sub-color = "#FFFFFF"
 sub-use-margins = yes
 sub-border-size = 1.5

 sub-ass-force-margins = yes

另外附上官方的wiki，我也是写这篇博客去看了才知道能临时加载字体，之前就在担心看外挂字幕的看多了会不会导致字体库太庞大：2_设置文件夹（便携式） · hooke007/MPV_lazy Wiki · GitHub（本来是想把这一页全弄过来的，不过想了想，我用mpv也是基于这个项目，要是这个项目没了，那我这个笔记也没啥用了，所以放原地址吧）

至于高清片源和字幕去哪儿找，反正是我个人笔记，浏览器收藏了就不记了

听歌

众所周知Windows最好的音乐播放器是foobar，视频播放器用最好的，音乐播放器自然也要用最好的，视频播放器用懒人包，音乐播放器自然也要用懒人包：GitHub - dream7180/foobox-cn: DUI 配置 for foobar2000

上面的项目以前是懒人包，现在是汉化版的增补包，如果想要究极懒人版也可以在项目里找到发布的博客，下载plus版就是懒人包了：asionwu - 博客园

（p.s.本来想传图的不过好懒，我发现纯语言描述也能描述出来……）

至于从网易云/QQ音乐下载的歌，我浏览器收藏了音乐解锁的项目（之前竟然被GitHub给DMCA Takedown了……还好又找到了大本营）

下载的歌曲部分会缺失信息，这些用软件自动或者手动修改就行了，我用的是MusicTag音乐标签，酷安有安卓版，里面能找到PC版，这玩意儿是真好用，扫描各个音乐APP匹配标签。备选用[picard](https://picard.musicbrainz.org/ 和)

（顺带一提，从Foobar2000可以直接拖歌到picard里）

啊本来写完了，突然想起来无损这回事儿，总之就是foobar设置里输出设备选为声卡对应的WASAPI或者ASIO可以独占输出，设置里也有DSD输出的相关设置，一般设置为DSD+PCM就行，如果要独占输出的话。具体我也本地存了pdf，就不写笔记里了。

突然想起来有个地方应该配个图，就是foobar默认双击打开某个音乐会清理上次的播放列表，应该指定一个双击打开音乐后加入的列表，这样之前自定义的列表就不会清零了：（“总是发送新文件到播放列表：”）
